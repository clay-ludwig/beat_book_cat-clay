# Talbot County Beat Book Viewer

An interactive beat book that connects synthesized content from Star Democrat newspaper articles to their original sources using semantic embeddings.

## How to Run

1. **Open Terminal** and navigate to the `Step 6 - Final Beat Book` folder

2. **Start a local web server**:
   ```bash
   python3 -m http.server 8000
   ```

3. **Open your browser** and go to:
   ```
   http://localhost:8000/viewer.html
   ```

4. **To stop the server**, press `Ctrl+C` in Terminal
